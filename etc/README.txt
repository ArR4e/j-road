This is X-tee library for Java platform, distributed under APL 2.0

Library jars have external dependencies that have not been included in this archive, these 
are listed in each jar's dependency descriptor (look in jar:META-INF/maven folder). Most
of these dependencies are also present in the sample web application archive, in the 
WEB-INF/lib folder.

Some Maven repositories where these external dependencies can be downloaded are:
  http://mirrors.ibiblio.org/pub/mirrors/maven2
